Who Am I And What Am I About?

My name is Maeson Williams. I'm currently a sophomore at NDSU working on my major in Computer Science. I was born in Logan, Utah, but grew up here in Fargo for most of my life. I started programming while attending West Fargo Highschool, and ever since then, I've enjoyed the challenges with programming, and the problem solving required for it. I've programmed in Java and C#, as well as have experience with HTML, CSS, Bootstrap and Github. I have a little experience in both Python and Javascript. I've used IDEs such as Eclipse, NetBeans, VS Code and Visual Studio. Some projects I've worked on in school includes this website and a web application for a fictitious music store. On my free time I like to occasionally make small games using either Unity or Libgdx. I've also dabbled in creating a web scrapper for Wikipedia, adding plugins to Minecraft, as well as modifying the inline Javascript of HTML built games. I want to go into this industry because I love working with computers, programming things and challenges that require me to think and problem solve.
